books app

  • GET /books - get list of all books
  • POST /books - create book
    Example:
    {"title":"First test book", "date":"2001-01-01", "author": "First test author", "description":"Best first test book", "image":"test_book1.jpg"}
    Response:
    {
    "id": 41
    }
  • PATCH /books - modify book by id
    Example:
    {"id": 31, "date":"2000-01-01", "title":"AAAA test book", "author": "AAAA test author", "description":"AAAAA first test book", "image":"AAA_book1.jpg"}
    Response:
    {
    "id": 31,
    "title": "AAAA test book",
    "date": "1999-12-31T22:00:00.000Z",
    "author": "AAAA test author",
    "description": "AAAAA first test book",
    "image": "AAA_book1.jpg"
    }
